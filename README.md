## 📌 Project Overview

This project is a fully functional Music Player web application built using HTML, CSS, and JavaScript as part of my Frontend Developer Internship task 3 at CodeAlpha.

The application provides interactive audio controls, real-time updates, and a clean, responsive interface designed for an enhanced user experience.

---

## 🚀 Features

### Core Functionalities

- Play and Pause controls  
- Next and Previous track navigation  
- Display of song title and artist  
- Real-time song duration  
- Interactive progress bar  
- Volume control slider  

### Additional Enhancements

- Playlist support
- Like button to like any track
- Search bar
- History Option to show the previous record
- Liked Option to display all Liked tracks
- Autoplay functionality  
- Responsive layout  
- Smooth animations and transitions  

---

## ⚙️ Application Logic

The music player uses JavaScript to:

- Control audio playback
- Update the progress bar dynamically  
- Display song metadata (title, artist, duration)  
- Handle volume adjustments  
- Automatically move to the next track when a song ends

## 🎧 Application Workflow

When the application loads, the available audio tracks are displayed in a structured list format. The listener can browse the playlist and select any track of their choice.

Upon selecting a track, a dedicated player interface opens, presenting detailed information including:

- Track cover / icon  
- Song title  
- Artist name  
- Total duration  

The player provides complete playback controls such as:

- Play / Pause  
- Next track  
- Previous track  
- Volume adjustment  

The **Next** and **Previous** buttons enable seamless navigation between tracks in the playlist, ensuring a smooth listening experience. The interface dynamically updates the track information whenever a new song is selected.
